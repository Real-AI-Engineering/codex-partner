# Contributing

## How to Contribute

### Report Issues

- Codex CLI compatibility issues
- Workflow improvements
- Documentation gaps

### Suggest New Modes

Open an issue describing:
- The use case
- Proposed flow
- Why existing modes don't cover it

### Improve Documentation

- Fix errors or unclear instructions
- Add examples from real usage
- Update for new Codex CLI versions

### General

- Keep instructions actionable
- Include example commands and expected output
- Test workflows before submitting
